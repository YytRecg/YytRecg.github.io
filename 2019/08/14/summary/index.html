Project summary

八月 14, 2019

“Characstar” stands for the star of character recognition. It is a handy character recognition application. This APP features its minimalist design and practical recognition functions，which can be applied in both daily work and study, and become one of the most useful productivity tools in your life.

【Instruction】

  1. Upload one image to conduct the image recognition process. User can choose the image from photo album or directly shoot by tapping the photo button.
  2. Select the area contains characters which you want to recognize. Once an area was selected，this area should be confirmed by tapping the confirm button before further selections.（You can select one or more areas in one image)
  3. Tap the identification button. The results of character recognition will show on the textbox below. Users can edit the results according to their purpose.
  4. The final text can be saved in the phone.

【Function/Feature description】

  1. Recognition region selection
  2. Character recognition
  3. Efficient recognition（Process one or more recognition areas simultaneously）
  4. Recognition results redaction
  5. Text saving function
